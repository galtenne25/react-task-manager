# Task Manager

This is a simple Task Manager application built with React and Vite. The application allows users to manage their tasks efficiently.

## Project Structure

```
task-manager
├── src
│   ├── components
│   │   ├── Header.jsx
│   │   ├── TodoList.jsx
│   │   ├── TodoItem.jsx
│   │   └── Footer.jsx
│   ├── App.jsx
│   ├── index.css
│   └── main.jsx
├── index.html
├── package.json
├── vite.config.js
└── README.md
```

## Components

- **Header**: Displays the header of the application.
- **TodoList**: Contains the list of tasks.
- **TodoItem**: Represents a single task item.
- **Footer**: Displays the footer of the application.

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd task-manager
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm run dev
   ```

5. Open your browser and go to `http://localhost:3000` to see the application in action.
